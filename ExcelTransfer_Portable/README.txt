Excel Transfer Tool - Portable Version
======================================

SYSTEM REQUIREMENTS:
- Windows 10 or later (64-bit)
- Microsoft Excel installed (for COM automation)
- Access to network drives (L:/ drive for default paths)

HOW TO RUN:
1. Extract all files to a folder
2. Double-click ExcelTransfer.exe
3. The application will start immediately - no installation needed

FOLDER STRUCTURE:
- ExcelTransfer.exe - Main application
- *.dll - Required Qt libraries
- JSON/ - Mapping configuration files
- platforms/, styles/, etc. - Qt plugins

FEATURES:
- Monthly Transfer: Transfer data for selected months/years
- Fill All Months: Batch transfer from January to target month
- Individual Transfer: Manual cell-to-cell data transfer
- Rolling Transfer: Sequential month-by-month processing
- Mapping Management: Configure and manage data transfer mappings

DEFAULT PATHS:
- Source/Dest: L:/Cost control/Cost Control/Cost control
- Mappings are loaded from JSON/ folder

NOTES:
- Keep all files in the same folder
- Do not delete any DLL files or plugin folders
- JSON mapping files can be edited with a text editor
- The application requires Excel to be installed for COM automation
